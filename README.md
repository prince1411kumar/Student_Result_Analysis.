# Student Result Analysis

## Project Overview
This is a beginner-friendly Python project that analyzes student exam results using NumPy.

The project performs basic data analysis such as:
- Total marks
- Average marks
- Highest marks
- Lowest marks
- Pass/Fail count

## Technologies Used
- Python
- NumPy

## Features
- Analyze student marks
- Calculate average score
- Find highest and lowest marks
- Display pass and fail statistics

## Project Structure
student-result-analysis/
│── student_result_analysis.py
│── StudentsPerformance.csv

## How to Run

1. Clone the repository
2. Install NumPy

```bash
pip install numpy
```

3. Run the project

```bash
python student_result_analysis.py
```
